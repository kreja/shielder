* 鼠标移上去变透明
* 快捷键 `CMD` + `T` 变透明
* 选择背景色

# 启动
* 根目录下执行 `npm install`
* 把根目录下的 app.icns 放到 `node_modules/nw/nwjs/nwjs.app/Contents/Resources` 下，这个是图标啦
* 进入 shielder 目录执行 `bower install`
* 回到根目录执行 `npm start` ，然后遮挡器就傻傻地出来了（⌒▽⌒ゞ
